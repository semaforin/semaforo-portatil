Diseñar el Semáforo con Arduino

El diseño comienza conectando cada LED a un pin digital del Arduino mediante resistencias. El LED rojo se conecta al pin digital 12 y el LED verde al pin 11. Cada LED está controlado individualmente por el Arduino, lo que permite gestionar las secuencias del semáforo.

El botón se conecta al pin digital 2, permitiendo la interacción manual para cambiar el estado de las luces. Además, un panel solar puede integrarse para hacer el sistema autónomo en términos de energía.

Muy Poca Seguridad en los Estudiantes

Para analizar la falta de seguridad vial entre los estudiantes, primero identificamos los problemas específicos como el comportamiento de los estudiantes y el estado de las vías cercanas a la escuela. Luego, revisamos la infraestructura, asegurándonos de que las calles y señales estén en buen estado. Observamos cómo los estudiantes interactúan con el tráfico y les preguntamos sobre sus hábitos de tránsito. También analizamos si hay suficiente educación y concienciación sobre seguridad vial en la escuela y si se están aplicando medidas efectivas para mejorar la seguridad.

Programación

Con la ayuda de placas de desarrollo como Arduino se controla el Display para que muestren señales tanto a los conductores como a los estudiantes que están próximos a cruzar, simulando el funcionamiento de un semáforo real.

Circuitos

Para el semáforo, el circuito principal se basa en un botón que controla las luces. El botón está conectado al Arduino y permite cambiar entre las diferentes luces del semáforo (rojo y verde). Cuando se presiona el botón, el Arduino enciende la luz correspondiente. No se utilizan temporizadores en este diseño; el cambio de luces se controla manualmente con el botón.

Microcontrolador

El microcontrolador utilizado en este proyecto es un Arduino Uno. Este dispositivo es el cerebro del semáforo, ya que se encarga de controlar las luces según el estado del botón y la pantalla del Display. El Arduino Uno procesa las señales del botón y enciende o apaga los LEDs (rojo y verde) en respuesta. Tiene pines digitales para conectar los LEDs y el botón, y es programable a través del entorno de desarrollo de Arduino para definir cómo deben comportarse las luces del semáforo.

Materiales

Arduino Uno: El microcontrolador principal encargado de gestionar las señales del semáforo.

LEDs: Dos LEDs (rojo, y verde) que simulan las luces del semáforo.

Resistencias: Protegen los LEDs de sobrecargas.

Botón de Control: Permite cambiar el estado del semáforo manualmente.

Panel Solar: Para alimentar el sistema de manera sostenible.

Cables de Conexión: Para interconectar todos los componentes.

Display: Muestra ilustraciones que proporcionan señales claras para el cruce de peatones, detenimiento de vehículos y al revés.

Cables de Conexión

Cables de Alimentación y Cables de Conexión

Cables de Alimentación: Estos cables son esenciales para suministrar energía al Arduino y a los componentes del semáforo. Utilizamos cables de alimentación para conectar el panel solar o una fuente de energía externa al Arduino, asegurando que todos los componentes funcionen correctamente. Estos cables garantizan que el sistema esté energizado de manera continua y estable.

Cables de Conexión para LEDs y Botón: Los cables de conexión se emplean para interconectar los LEDs (rojo y verde) y el botón con el Arduino. Utilizamos cables de puente para conectar cada LED a los pines digitales del Arduino a través de resistencias. El botón también se conecta a un pin digital del Arduino mediante cables de conexión, permitiendo así el control manual del semáforo. Estos cables aseguran una conexión fiable y ordenada entre el Arduino y los componentes del circuito.
